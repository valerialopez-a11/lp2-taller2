## Instalación
1. Clonar el proyecto
```bash
git clone https://github.com/UR-CC/lp2-taller2.git
```

2. Crear y activar entorno virtual
```bash
cd lp2-taller2
python -m venv venv
venv/bin/activate
```

3. Instalar librerías y dependencias
```bash
pip install -r requirements.txt
```
    
## Ejecución
1. Ejecutar el proyecto
```bash
cd lp2-taller2
python app.py
```
